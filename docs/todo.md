# Admin Guide

## Installation

- Clone repository
- uv sync
- Copy over example.env to .env
- Customize .env (especially the ENVIRONEMNT)
- uv run cli stack-deploy

## Updates

- Jobs Overview - nothing is running
- Maintenance mode on
- Go to production folder
- uv run cli db-backup
- uv run cli stack-rm
- git pull origin main
- Compare example.env and .env (update .env)
- uv run cli compose-pull
- uv run cli stack-deploy
- Turn off maintenance mode
